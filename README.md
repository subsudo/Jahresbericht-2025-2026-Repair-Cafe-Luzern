# Jahresbericht 2025/2026 Repair Café Luzern

Statische HTML-Site fuer den oeffentlichen Jahresbericht 2025/2026.

## Inhalt

- `index.html` ist die Startseite fuer GitHub Pages.
- `.nojekyll` verhindert, dass GitHub Pages Dateien oder Ordner speziell behandelt.
- Enthalten sind nur die fuer die Website benoetigten Assets.
- Arbeitsdateien, Rohdaten, Scans, Protokolle, Budgets, Praesentationen und Skripte sind bewusst nicht enthalten.

## Veröffentlichen mit GitHub Pages

1. Inhalt dieses Ordners in die Repo `subsudo/Jahresbericht-2025-2026-Repair-Cafe-Luzern` legen.
2. Auf `main` pushen.
3. In GitHub unter `Settings > Pages` aktivieren:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

Danach ist die Site voraussichtlich erreichbar unter:

`https://subsudo.github.io/Jahresbericht-2025-2026-Repair-Cafe-Luzern/`

## Vor Public-Schaltung prüfen

- Bildrechte und Einverständnis der erkennbaren Personen.
- Logos und Partnernennungen.
- Darstellung auf Beamer/Desktop und Smartphone.
- Keine internen Dateien versehentlich im Repo.
